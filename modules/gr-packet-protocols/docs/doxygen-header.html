About gr-packet-protocols

gr-packet-protocols is a comprehensive GNU Radio Out-of-Tree (OOT) module that provides implementations of various packet radio protocols commonly used in amateur radio and digital communications.

AX.25 Protocol

Complete implementation of the AX.25 (Amateur X.25) protocol with support for I, S, and U frame types, addressing, and KISS TNC interface.

FX.25 Forward Error Correction

Reed-Solomon based Forward Error Correction for AX.25 frames, providing reliable transmission over noisy radio channels.

IL2P Protocol

Modern Improved Layer 2 Protocol implementation with advanced features for contemporary packet radio systems.

KISS TNC Interface

Terminal Node Controller interface supporting USB CDC and TCP/IP connections for hardware integration.
